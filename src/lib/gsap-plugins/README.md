# GSAP Bonus Plugins

This directory should contain the following files:

- `ScrollSmoother.js` (or `ScrollSmoother.min.js`)
- `SplitText.js` (or `SplitText.min.js`)

## How to get these files:

1. Visit https://greensock.com/docs/v3/Plugins/ScrollSmoother
2. Click the download button to get `ScrollSmoother.js`
3. Visit https://greensock.com/docs/v3/Plugins/SplitText  
4. Click the download button to get `SplitText.js`
5. Place both files in this directory

**Note:** These plugins are now FREE as of April 30, 2025. You just need to download them from the GSAP website.
